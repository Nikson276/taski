Открыть файл конфигурации Nginx
    sudo nano /etc/nginx/sites-enabled/default

Проверка файла конфигурации Nginx
    sudo nginx -t

Перезагрузка конфига Nginx
    sudo systemctl reload nginx

Управление Nginx:
    sudo systemctl start nginx
    sudo systemctl restart nginx
    sudo nginx -s сигнал
        stop — быстрое завершение
        quit — плавное завершение
        reload — перезагрузка конфигурационного файла
        reopen — переоткрытие лог-файлов

Просмотр логов Nginx
    sudo tail <путь и имя_файла>
    sudo tail /var/log/nginx/access.log
    sudo tail /var/log/nginx/error.log

    Перед именем файла есть необязательный флаг -n число_строк (берет опр кол-во строк с конца файла)
